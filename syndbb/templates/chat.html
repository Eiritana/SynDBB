Our chat system runs on the Matrix Protocol:
  • Matrix provides state of the art end-to-end encryption in beta using the Olm and Megolm cryptographic ratchets, and ensuring that only explicitly authorized devices can participate in a conversation. Based on the Double Ratchet Algorithm popularised by Signal.
  • Supports group chats with hundreds of devices via the new Megolm shared ratchet, with either perfect forward secrecy or the ability to decrypt conversation history on new devices.
  • Create and manage fully distributed conversations with no single points of control or failure.
  • Real-time synchronised history and state across all clients.
  • WebRTC VoIP/Video calling using Matrix signalling.
  • In the future, all channels and conversations will have encryption enabled by default.
More Info

Why Matrix:
  • All messages are encrypted using end-to-end encryption.
  • Chat history is inaccessible to users without the decryption key and is cleared regularly.
  • Users can create and moderate their own channels.
  • Get information about events and site activity.
  • Full integration with D2K5
