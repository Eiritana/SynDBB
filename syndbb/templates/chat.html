Network Information & Commands

The D2K5 Network has its own official IRC bot called Spunky, which provides many useful commands and is also used for identifying users between the site and the IRC.

General:
  • !info [user] - Look up information about a user on D2K5.
  • !calc <calculation> - Calculate something.
  • !wa <calculation> - Calculate something using Wolfram Alpha.
  • !dice <XdY> - Rolls dice using either the XdY or XdYvZ format.
  • !yt <terms> - Find a video on YouTube.
  • !ddg <terms> - Search something on DuckDuckGo.
  • !wiki <terms> - Search something on Wikipedia.
  • !ety <word> - Look up the etymology of a word.
  • !define <word> - Look up the definition of a word.
  • !countdown <YYYY MM DD> - Countdown to a date.
  • !ip <IP> - Look up information of an IP address.
  • !isup <url> - Check if a website is up.
  • !tr <message> - Translates the given message into English.
  • !rand [int1] [int2] - Generates a random number.
  • !settz <timezone> - Set your preferred timezone.
  • !settf <format> - Set your preferred time format.
  • !setlocation <location> - Set your preferred location.
Moderation:
  • !up - Request a higher rank (>Half-Op).
  • !down - Lowers your rank to voice (Member<).
  • !ban <user> - Bans a user.
  • !unban <user> - Unbans a user.
  • !kick <user> - Kicks a user.
  • !kb <user> - Kicks and bans a user.
  • !quiet <user> - Silences a user.
  • !unquiet <user> - Unsilences a user.
  • !topic <message> - Changes the topic of the current channel.
  • !channeltz <timezone> - Sets the default timezone.
  • !setctf <format> - Sets the default time format.
  • !seen <user> - Shows when and where the user was last seen.
